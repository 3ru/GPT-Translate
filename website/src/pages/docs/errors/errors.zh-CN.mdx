# 错误处理

## 机器人错误

### `You have no permission to use`

此机器人限制了通过命令等执行翻译的用户只能是仓库的**Write**权限用户。
OpenAI API是按使用付费的，我设置了这个限制，以保护我的软件用户免受恶意用户的侵害。

如果你想给任何用户执行命令的权限，请邀请他们成为仓库的**collaborator**。

## API错误

### `503` Error

可能已达到对OpenAI API的调用次数限制。请稍后再试。
对于根本解决方案，[需要帮助](https://github.com/3ru/gpt-translate/issues/18)。
