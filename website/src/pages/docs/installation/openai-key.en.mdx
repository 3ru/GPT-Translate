## Key Generation

<Steps>

### Go to the [API Keys page](https://platform.openai.com/account/api-keys).

![API Keys page](https://i.gyazo.com/5acc5d6417413a2f34fc52075bd52b74.png)

### Create a new API Key

Click the `Create new secret key` button.
![Create new secret key](https://i.gyazo.com/b2797f624de75fb28e1732ead38f2676.png)

Enter the name of the API Key and click the `Create secret key` button.
![secret key name input](https://i.gyazo.com/bfc3c62a78de701b53c2f05a37df32bb.png)

### Copy the API Key.

This key will be used in the next GitHub repository setup. Do not close the tab or make a temporary note of it somewhere.
![secret key](https://i.gyazo.com/b1ab2746efe410e0d25c530b150f94bb.png)

</Steps>
