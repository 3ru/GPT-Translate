## 密钥生成

<Steps>

### 转到 [API密钥页面](https://platform.openai.com/account/api-keys)。

![API Keys page](https://i.gyazo.com/5acc5d6417413a2f34fc52075bd52b74.png)

### 创建一个新的API密钥

点击 `创建新的密钥` 按钮。
![Create new secret key](https://i.gyazo.com/b2797f624de75fb28e1732ead38f2676.png)

输入API密钥的名称，然后点击 `创建密钥` 按钮。
![secret key name input](https://i.gyazo.com/bfc3c62a78de701b53c2f05a37df32bb.png)

### 复制API密钥。

这个密钥将在下一个GitHub仓库设置中使用。不要关闭这个标签页，或者在某个地方临时记下它。
![secret key](https://i.gyazo.com/b1ab2746efe410e0d25c530b150f94bb.png)

</Steps>
